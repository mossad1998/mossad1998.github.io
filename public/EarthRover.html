Earth Rover

Earth rover is differential robot which can be controlled through the internet using ssh connection. the robot is equiped with two cameras allowing it to perform vision tasks, e.g. face recognition. it is also able to detect fire and warn the residenents with both sound alarm and e-mail.

  • GitHub
Earth rover software consists of two major sub-systems interacting with each other in real time:
--->Master sub-system written in python and runs on raspberry pi 3.
--->Slave sub-system wriiten in C++ and runs on Tiva-C ARM Cortex-M4F based.

Master sub-system main tasks:
1. Receive control commands as str through SSH port.
2. Parse the control commands and pass it to the slave sub-system though UART protocal.
3. Run and maintain face recognition service.
4. Run and maintain fire detection service.
5. Run and maintain the warnning and e-mail services.

Slave sub-system main tasks:
1. Recieve the parsed control commands from the master sub-system though UARD protocal and reply with ack feedback.
2. Generate the required PWMs to run the motors.
3. Determine and log the position of the robot using wheels encoders and IMU sensors.
4. Sense the surrounding environment using infrared sensors.
