Earth Rover

Earth rover is differential robot which can be controlled from any place in the world using ssh. the robot has two cameras which give the robot the abilities of face recognition and fire detection. the robot can sense if there is something wrong and ping you through an email.

  • GitHub
The robot consists of two levels which are:
---> High level which is pythons scripts run on raspberry pi 3.
---> Low level which is c++ files run on Tiva c ARM Cortex-M4F based.

High level main tasks:
1. Run the manual control script which can be executed using ssh.
2. The manual control script communicates with the tiva c using UART protocal to send different commands.
3. Executing face recognition script.
4. Executing the fire detection script.
5. Executing the ping script which can send the used an email.

Low level main tasks:
1. Communicating with the master board using UART protocal.
2. Generating the required PWMs to run the motors.
3. Determine the position of the robot using wheels encoders and IMU.
4. Sensing the surroundedenvironment using infrared sensors.
