Remotely Operated Vehicle (ROV)

This project focused on building a Remotely Operated Vehicle (ROV) to compete in the MATE International ROV Competition in the U.S. I was responsible for designing and implementing both the hardware kit and the Telematics Control Unit (TCU), which formed the backbone of the vehicle’s embedded systems and operator interface.

Telematics Control Unit (TCU) – Core Components

  1. Power Subsystem: Delivered up to 1440 W to support motors, control electronics, and auxiliary payloads.
  2. Control Console: Integrated joystick, operational toggles, and a hardware-based kill switch for immediate safety shutdown.
  3. Pilot Interface: Multi-screen station displaying real-time video streams and telemetry dashboards (depth, power usage, and sensor data).

Hardware Kit – Embedded & Networking Functionalities

  • Designed a DC-DC power stage to step down 48V tethered supply to regulated 12V rails for ESCs, servos, and MCUs.
  • Implemented power distribution bus with current monitoring and protection circuitry for reliable operation.
  • Re-transmitted live video feeds from multiple cameras via Ethernet (UDP streaming) to the TCU operator station.
  • Designed an Ethernet-based control protocol to send command frames and receive feedback/status packets in real time.
  • Integrated embedded controllers (MCUs) for actuator control and sensor fusion, interfacing via UART, PWM, and SPI buses.

ROV Missions Successfully Executed

1. Used simulated flight data to determine wreckage search zones.
2. Removed debris from a submerged engine using lift bags.
3. Transported and returned the engine to the surface using buoyancy-assisted mechanisms.
4. Processed tidal data and nautical charts to determine the optimal turbine installation site.
5. Installed a mock tidal turbine at the identified optimal location.
